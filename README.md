# v0-weather-web-app

A simple, clean, and fast weather app built with Next.js and TypeScript. It lets you search for any city and instantly check the temperature, humidity, wind speed, and other basic conditions. The project is lightweight, beginner-friendly, and easy to deploy.

> **Live Demo:** https://v0-weather-web-app-yobo.vercel.app/
